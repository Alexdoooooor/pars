Доступ только для администраторов

API и консоль защищены Basic Auth. Публичный сайт показывает статус сервиса, но не выдаёт результаты сравнения.

🧭

Развёртывание в подкаталоге

Для адреса вида /pars/ задайте APP_BASE_URL=/pars и проксируйте весь путь к uvicorn (см. deploy/nginx-aisidora-pars.example.conf).

🧩

Режим парсера

PARSER_MODE=live включает HTTP-сбор цен (best-effort), PARSER_MODE=mock оставляет детерминированные тестовые значения.

Публичная витрина без демо-результатов. Сравнение цен — в защищённой консоли.
